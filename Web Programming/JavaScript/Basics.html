Variables in JS

3 ways to store the data in JS:

  1. var keyword:Old school of creating variable
  2. let keyword:New way of declaring variables and said to be powerfull
  3. const keyword:For storing data which does not change
JS does not any data type in it.Internally it differentiate the types of data as number,string,boolean,object and undefined.
